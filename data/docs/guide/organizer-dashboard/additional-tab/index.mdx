<Callout title="Mode Specific Tabs" type="info">
  The following additional tabs are available only in <strong>Offline</strong> or <strong>Online with Application Review</strong> mode.
</Callout>

These tabs give organizers more control over in-person or hybrid hackathons by enabling them to:

- Track travel reimbursements
- Manage RSVP responses
- Handle check-ins
- Access logistics like t-shirt sizing and dietary preferences

Explore each of these tabs in the respective sections below.
