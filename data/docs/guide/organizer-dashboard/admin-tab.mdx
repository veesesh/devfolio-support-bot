---
title: Admin Tab
description: The Admin tab contains essential features that make it easier to manage your hackathon.
---

import { Tabs, Tab } from "fumadocs-ui/components/tabs";

<Tabs items={["Search Team and Hackers", "Change & Extend Deadlines", "Download CSV", "Make an Announcement"]} defaultValue="Search Team and Hackers">
  <Tab value="Search Team and Hackers">
    <p>All relevant data of hackers is visible through the dashboard itself.</p>
    <img src="/assets/admin-1.gif" alt="Search Hackers in Admin Tab" />

    <Callout title="Sensitive Data Access" type="warning">
      Certain sensitive information such as phone numbers is visible upon entering the username of the hacker in the search bar, which is why <strong>access to this tab must be provided with discretion</strong>.
    </Callout>

    <p>You can also extend the RSVP deadline for a specific user by X number of days or hours. This is useful when a user misses the RSVP deadline after being accepted but you still want them to attend.</p>
    <img src="/assets/image-9.12.webp" alt="Extend RSVP deadline" />
  </Tab>

  <Tab value="Change & Extend Deadlines">
    <h3>Change Application Deadline</h3>
    <img src="/assets/image-9.14.webp" alt="Change Application Deadline" />

    <h3>Extend RSVP Deadline</h3>
    <p>You can extend the RSVP deadline for all hackers using the dashboard. For example, if you extend the RSVP Deadline by <code>N</code> days, hackers can RSVP within <code>N</code> days from the moment you apply the extension.</p>
    <img src="/assets/image-9.15.webp" alt="Extend RSVP Deadline" />

    <h3>Extend Project Submission Deadline</h3>
    <p>If you need to give hackers more time due to unforeseen circumstances, you can extend the project submission deadline from here.</p>
    <img src="/assets/image-9.16.webp" alt="Extend Project Deadline" />
  </Tab>

  <Tab value="Download CSV">
    <h3>Online Mode</h3>
    <p>Download the list of participants who submitted projects to your hackathon.</p>
    <img src="/assets/image-9.17.webp" alt="Download participants - online mode" />

    <h3>Online with Application Review / Offline Mode</h3>
    <p>Download the list of participants who RSVP’d to your hackathon.</p>
    <img src="/assets/image-9.18.webp" alt="Download RSVP list" />
  </Tab>

  <Tab value="Make an Announcement">
    <p>The Admin tab also lets you send announcements to hackers — to keep them informed about updates or changes. When clicking on ‘Preview’, a checkbox will appear that allows the announcement to also be sent via email.</p>
    <img src="/assets/image-9.19.webp" alt="Audience Segments" />

    <Callout title="Tip" type="info">
      If you wish to send out the announcement as an email (recommended), make sure you check the option to <code>Also send as email</code> before hitting send.
    </Callout>

    <img src="/assets/image-9.20.webp" alt="Send Announcement Preview" />
  </Tab>
</Tabs>
